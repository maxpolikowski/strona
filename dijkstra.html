Dijkstra

Powrót do strony głównej

Algorytm Dijkstry działa poprawnie tylko w grafach, w których wszystkie wagi krawędzi są nieujemne. Graf może być skierowany lub nieskierowany.

Algorytm Dijkstry służy do znajdowania najkrótszych ścieżek w grafie z nieujemnymi wagami krawędzi.

Przykładowy graf

Inicjalizacja

Tworzymy tablicę dist, gdzie dist[v] oznacza najkrótszą znaną odległość z wierzchołka startowego do wierzchołka v.

  • dist[1] = 0
  • dist[v] = ∞ dla pozostałych wierzchołków

Dodatkowo korzystamy z kolejki priorytetowej, która umożliwia szybki wybór wierzchołka o najmniejszej aktualnej wartości dist.

Działanie algorytmu krok po kroku

Krok 1 – wierzchołek 1
Z wierzchołka 1 możemy przejść do wierzchołków 3, 2 oraz 5.
Krok 2 – wierzchołek 3
Odległość do wierzchołka 2 nie ulega poprawie.
Krok 3 – wierzchołek 2
Aktualizujemy odległość do wierzchołka 6.
Krok 4 – wierzchołek 5
Aktualizujemy odległości do wierzchołków 4 oraz 8.

Relaksacja krawędzi

Sprawdzamy warunek:

dist[u] + w(u, v) < dist[v]

Jeśli warunek jest spełniony, aktualizujemy wartość dist[v].

Zakończenie

Po zakończeniu działania algorytmu tablica dist zawiera najkrótsze odległości z wierzchołka 1 do wszystkich pozostałych wierzchołków.

Kod dijkstry


#include <iostream>
#include <queue>
#include <vector>

using namespace std;

const int MAXN = 100000;
vector<pair<int, int>> graf[MAXN];
int dist[MAXN];

int main() {

    int n, m;

    priority_queue<pair<int, int>> kolejka;
    kolejka.push({0, 1});

    while (!kolejka.empty()) {
        int v = kolejka.top().second;
        int cost = -kolejka.top().first;
        kolejka.pop();

        if (dist[v] < cost) continue;

        for (auto &[s, waga] : graf[v]) {
            if (cost + waga < dist[s]) {
                dist[s] = cost + waga;
                kolejka.push({-(cost + waga), s});
            }
        }
    }

    return 0;
}
